O que são Sistemas Operacionais?

Sistema operacional é o programa que permite a comunicação do usuário com os componentes físicos do computador (denominados como hardwares) através de uma interface. Ou seja, além de executar e oferecer recursos para softwares e aplicativos, também coordena os recursos de hardware do computador.

Tipos de Sistemas Operacionais

  • Monotarefa: Permite que o processador realize apenas uma tarefa por vez;
  • Multitarefa: Permite que o processador consiga realizar várias tarefas; simultaneamente;
  • Monousuário: Possibilita que apenas um usuário use o computador;
  • Multiusuário: Possibilita de vários usuários utilizem o computador ao mesmo tempo;
