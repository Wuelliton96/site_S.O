Organização de pastas

Disco Local (C:)

A unidade C é considerada a unidade de disco rígido principal do sistema e é usada para armazenar o sistema operacional, arquivos do sistema e outros aplicativos e seus arquivos relacionados.

Usuários

Nos Windows 7 e Vista, a pasta “Documents and settings” passou a se chamar “Usuários”. Nela você encontra diretórios para todas as contas criadas no seu Windows. Ao acessá-las você confere as pastas padrão para músicas, imagens, vídeos e outros documentos.

Arquivos de programas (Program Files)

Aqui é onde praticamente todos os programas instalados no computador são inseridos. Ao acessar esta, você observa que há dezenas de pastas com os nomes dos aplicativos ou de seus desenvolvedores, o que inclui quase tudo o que é necessário para que os programas funcionem.

Windows

Esta pasta é a mais importante do computador, pois nela se encontra toda a estrutura do sistema operacional. É importante que você nunca apague nada dentro dela ou mexa em seus arquivos e pastas, a menos que tenha muito conhecimento na área.

Comandos para navegação e manipulação de pastas

  • MD : Criar uma Nova pastas
    • C:\Users\User\Pessoa> md Crianca
  • DIR : Listando arquivos e pastas
    • C:\Users\User\Pessoa> dir
    Pasta de C:\User\User\Pessoa
                    27/04/2022  20:02    DIR          .
                27/04/2022  20:02    DIR          ..
                27/04/2022  20:02    DIR          Crianca
                27/04/2022  19:57    DIR          Fisica
                27/04/2022  19:57    DIR          Juridica
                            0 arquivo(s)          0 bytes
                            5 pasta(s)   25.186.639.872 bytes disponívei
  • MOVE : Mover arquivos ou renomear pastas
    • C:\Users\User\Pessoa> move Crianca Fisica
  • CD : Entrar em uma pasta
    • C:\Users\User\Pessoa> cd Crianca
  • CD.. : Retornar para a pasta anterior
    • C:\Users\User\Pessoa\Crianca> cd..
  • RD : Remover pastas
    • C:\Users\User\Pessoa> rd Crianca

Comandos para navegação e manipulação de arquivos

  • COPY : Copiar arquivos [copy arquivo destino]
    • C:\Users\User\Pessoa\Fisica> copy cpf.jpg C:\Users\User\Pessoa
  • MOVE : Mover arquivos [move arquivo destino]
    • C:\Users\User\Pessoa> move cpf.jpg C:\Users\User\Pessoa\Fisica
  • ERASE : Deletar arquivos [erase arquivo]
    • C:\Users\User\Pessoa\Fisica> erase cpf.jpg
  • DEL : Deletar arquivos [erase arquivo]
    • C:\Users\User\Pessoa\Fisica> del cpf.jpg
  • REN : Renomear arquivos [ren nomeArquivo novoNomeArquivo]
    • C:\Users\User\Pessoa\Fisica> ren cpf.jpg cpf2.jpg
  • ASSOC : Exibe ou modifica associações de extensões de arquivo
    • C:\Users\User\Pessoa\Fisica>assoc .jpg 
    .jpg=jpegfile
  • ATTRIB : Exibe ou altera atributos de arquivos
    • C:\Users\User\Pessoa\Fisica>attrib /? 
     
                Exibe ou altera atributos de arquivo.
                
                ATTRIB [+R | -R] [+A | -A] [+S | -S] [+H | -H] [+O | -O] [+I | -I] [+X | -X] [+P | -P] [+U | -U]
                        [unidade:][caminho][nome_de_arquivo] [/S [/D]] [/L]
                
                  +   Define um atributo.
                  -   Limpa um atributo.
                  R   Atributo de arquivo somente leitura.
                  A   Atributo de arquivo morto.
                  S   Atributo de arquivo do sistema.
                  H   Atributo de arquivo oculto.
                  O   Atributo offline.
                  I   Atributo de arquivo sem conteúdo indexado.
                  X   Sem atributo de arquivo de remoção.
                  V   Atributo de integridade.
                  P   Atributo fixado.
                  U   Atributo desafixado.
                  B   Atributo de Blob SMR.
                  [unidade:][caminho][nome_de_arquivo]
                      Especifica um ou mais arquivos para o atributo processar.
                  /S  Processa os arquivos correspondentes na pasta atual
                      e em todas as subpastas.
                  /D  Inclui pastas no processamento.
                  /L  Trabalha nos atributos do Link Simbólico versus
                      o destino do Link Simbólico
  • REPLACE : Substitui arquivos
    • C:\Users\User\Pessoa>replace /?
                Substitui arquivos.
                
                REPLACE [unidade1:][cam1]arquivo [unidade2:][cam2] [/A] [/P] [/R] [/W]
                REPLACE [unidade1:][cam1]arquivo [unidade2:][cam2] [/P] [/R] [/S] [/W] [/U]
                
                  [unidade1:][cam1]arquivo Especifica o(s) arquivo(s) de origem.
                  [unidade2:][cam2]        Especifica a pasta onde os arquivos deverão
                                          ser substituídos.
                  /A                      Adiciona novos arquivos à pasta de destino.
                                          Não podem ser usados com as opções /S ou /U.
                  /P                      Pede confirmação antes de substituir um arquivo ou
                                          de adicionar um arquivo de origem.
                  /R                      Substitui arquivos de somente leitura e arquivos
                                          sem proteção.
                  /S                      Substitui arquivos em todas as subpastas
                                          da pasta de destino. Não pode ser usado com
                                          a opção /A.
                  /W                      Espera um disco ser inserido antes de começar.
                  /U                      Substitui (atualiza) somente arquivos mais antigos
                                          que os arquivos de origem. Não pode ser usado com
                                          a opção /A.
